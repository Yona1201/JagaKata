Panduan Penggunaan JagaKata

Ikuti langkah-langkah mudah berikut untuk mendeteksi ujaran kebencian menggunakan aplikasi kami.

Langkah 1: Pilih Mode Input

Buka halaman "Deteksi", lalu pilih salah satu dari tiga mode input yang tersedia di sidebar kiri sesuai dengan sumber data Anda: Teks, YouTube, atau File.

Langkah 2: Masukkan Data Anda

Setelah memilih mode, masukkan data Anda pada kolom yang tersedia. Anda bisa mengetik teks, menempelkan URL video YouTube, atau mengunggah file berformat .csv, .xlsx, atau .txt.

Langkah 3: Lihat Hasil Analisis

Klik tombol "Deteksi" dan tunggu beberapa saat. Aplikasi akan menampilkan hasil analisis dalam bentuk diagram ringkasan dan tabel rincian untuk setiap teks yang diproses.

Langkah 4: Unduh Hasil (Opsional)

Jika Anda membutuhkan salinan data hasil analisis, klik tombol "Download CSV" yang terletak di bawah tabel untuk menyimpan hasilnya ke komputer Anda.
